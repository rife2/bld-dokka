
## Compile the Example

```console
./bld compile
```

## Run the Example

```console
./bld run
```

## Run the Tests

```console
./bld test
```

## Build the documentation with [Dokka](https://github.com/Kotlin/dokka)

```console
./bld docs

./bld javadoc
./bld dokka-html
./bld dokka-gfm
./bld dokka-jekyll
```
